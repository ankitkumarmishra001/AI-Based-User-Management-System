# AI-Based User Management System

An AI-powered User Management System built using Django and React.

## Tech Stack

- Backend: Django
- Frontend: React + Vite
- Database: SQLite
- Authentication: JWT
- AI: Gemini/OpenAI API

## Features

- User Registration
- Login
- JWT Authentication
- Dashboard
- AI Chatbot
- User Profile
- Security Check

## Installation

### Backend

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
